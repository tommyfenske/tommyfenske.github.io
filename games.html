Rotate!:


A short, arcade style game I made using the Pico-8 game engine. Rotate! has steep learning curve and uprgade mechanics insprired by roguelike games. Rotate and shoot at the incoming enemies to level up and score points. Try and beat my score 0f 1030!

[play]


Jetpack Joyride Demake:


I remade the popular mobile game Jetpack Joyride inside the game engine, Pico-8. Pico-8 only allows 16 colors, 4 sound channels, and 256 8x8 sprites, hence the name "De-make". I learned to code using Pico-8's version of Lua. The engine's limitations meant Object Oriented, abstract, and reusable code was a priority. I made all code, sounds, and sprites. My high score is 115!

[play]
